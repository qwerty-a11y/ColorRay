# 数据获取接口使用说明

## 1. 导入并调用函数

传入视频文件路径与处理帧数上限，函数将执行解码并返回包含结果数据的列表。

```python
from test_memory_pipeline import process_video_pipeline

# 调用接口获取数据列表
results = process_video_pipeline(video_path='test1.MOV', frame_limit=60)
```
